Harvestrolley Docs Last updated: 2022-11-09

Bu sayfa Harvestrolley kullanımı için gerekli dökümanları içerir.


						
			
						
				    

				    

						
Giriş

						
Harvestrolley adresine gidilir.

						
						

						
Email address kolonuna email adresi girilir.

						
Password kolonuna şifre girilir.

						
Remember me kutucuğunu işaretlerseniz bir sonraki girişinizde bilgileriniz otomatik ekrana gelir.

						
						
Şifrenizi unuttuysanız Forgot Password? yazısına tıklayınız.

						Log in butonuna tıklayarak giriş yapabilirsiniz.
						
                        
                        
                        
                        
                       
					

					
					

						
Forgot Password?

						
Aşağıdaki videodan şifrenizi nasıl sıfırlayacağınızı öğrenebilirsiniz.

						
Bu videodaki yapılan işlem mail adresinize şifre sıfırlama linkinizi gönderecektir.

						

						
						
						
						

						


					

					
					

						
Users

						
Users menüsünde yeni kullanıcı eklemek için New User butonuna tıklanır.

						
Açılan pencerede kullanıcının bilgileri girilir ve Select Role kısmından kullanıcıya uygun bir rol seçilir.

						
Bütün kolonlar doldurulduktan sonra Create User butonuna tıklanır.

                        
						

						
						
						
						

						
Filter menüsünde kullanıcıları belirli kriterlere göre filtreleyebilirsiniz.

						
Name,Surname,Email,Phone v.s gibi

						
İstenen filtreler uygulandıktan sonra Filter butonuna tıklanır.

						
						

						
						

						


						
Export butonuna tıklandığında Succesfully Exported yazısı ekrana gelecektir.

						
User Export dosyanızı almak için My Exports menüsüne gidebilirsiniz.

						
My Exports menüsünde Processing işlemi kısa bir süre sonra bitecektir.

						
Success olduktan sonra Excel olarak indirebilirsiniz.


						

						
						

						


						
Columns butonundan yaptığınız seçimlere göre kullanıcılar ekrana listelenir.


						

						
						

						
User Delete

						
Delete işlemi için Delete butonuna tıklanır.

						
Do you want to delete this item?

						
Sorusuna Delete dendiğinde kullanıcı sistemden silinir.



						

						
						

						


						
User Info

						
Users listesinden bir kullanıcıya tıklandığında bilgileri görüntülenen bir ekran açılır bu ekranda kullanıcıya not eklenebilir.

						
Bilgileri düzenlenebilir hatta kullanıcı silinebilir ayrıca kullanıcının logları,orderleri,adresleri görüntülenir.

						
Aşağıdaki video'da kullanımını öğrenebilirsiniz.


						

						
						

						


						
					

					
					

						
Roles

						
New butonuna tıklayıp Add Role penceresinden yeni rol oluşturabilirsiniz.

						
Name kolonuna rol ismi.

						
Scope kolonunda Admin>Warehouse>Store olarak 3 seçenek mevcut. Seçilen scope göre yetkiler ve panel değişmektedir.

						
Permissions kolonunda rolün yetkilerini seçiniz.

						
Aşağıdaki videoda izleyebilirsiniz.


						

						
						
						


						
Bir rol sonradan nasıl düzenlenir.


						

						
						
						

                        
						
Bir rol nasıl silinir.

Minimum Cart Total kolonundan minimum sepet tutarı ayarlanır.
